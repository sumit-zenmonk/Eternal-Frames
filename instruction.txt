Only to create graph images or .gv files
(https://manishbit97.medium.com/identifying-circular-dependencies-in-nestjs-using-madge-de137cd7f74f)
    Installation
        npm -g install madge
        sudo apt-get install graphviz #( On Ubuntu )

    Generate Dependency Graph
        madge --image graph.png --extensions ts ./backend/src

    Generate Circular Dependency Graph
        madge --image graph.png --extensions ts --circular ./backend/src/

    Generating .gv file using madge 
        madge --dot --extensions ts ./backend/src/ > graph.gv


Steps To Run This Repo Code (Open 2 Terminal)
0. -> create 3 duplicate envs in 3 places
1. -> docker compose up --build (1st terminal command)
2. -> docker compose exec backend sh (2nd terminal command)


For DB Migration
3. user Module -> npm run build && npm run migration:user:run && npm run seed:user:user
4. billing Module -> npm run build && npm run migration:billing:run && npm run seed:billing:user && npm run seed:billing:subscription
5. event Module -> npm run build && npm run migration:event:run && npm run seed:event:user
